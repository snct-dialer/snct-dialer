Thank you for choosing ViciPhone.

In order to use ViciPhone, upload the code inside this folder into a subdirectory from your Vicidial AGC directory (typically /srv/www/htdocs/agc/viciphone), then point your webphone URL to the absolute or relative path within your server, such as /agc/viciphone/viciphone.php.

If you are using ViciPhone in conjunction with Vicidial you will need to change this setting in ADMIN → System Settings.
